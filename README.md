# Done For Music - Synth & Audio Gear Knowledge Base

A curated reference library for a personal/studio collection of synthesizers, 
drum machines, mixers, and outboard gear. Designed to work alongside Claude for hands-free lookup and documentation.

## How it's used

- No longer browsing through manuals, just prompt what you need to look up.
- Prompt for optimal way to connect gear to the mixer or to each other
- Build different HW setups quickly, with details plans on audio and MIDI connections and how to setup mixer channels.

## What's in here

- **Device manuals** — original PDFs from manufacturers, never modified
- **Connectivity reference** (`connectivity_reference.md`) — audio and MIDI 
  patch points across all devices in one place
- **Errata files** — manufacturer corrections to specific manuals
- **Quirks files** — discovered bugs, edge cases, and workarounds not covered 
  by the official docs
- **CLAUDE.md** — instructions for the AI assistant on how to navigate this repo
